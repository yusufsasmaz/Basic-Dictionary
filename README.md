## How to Run

Clone the project

```bash
  git clone https://github.com/yusufsasmaz/Basic-Dictionary.git
```

Composer

```bash
  composer install
```
 `.env` after database connection

+ Install dependencies

```bash
  npm install
```

```bash
  npm run build
```

```bash
  php artisan migrate
```
+ Run
```bash
  php artisan serve
```
```bash
  npm run dev
```

+ Add Seeds

```bash
  php artisan db:seed --class=WordSeeder
```
